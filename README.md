# Dynamic-Pathfinding-Agent
How to run this code:
Install python extension on your computer
Open terminal on VS Code and write:
py -3.11 -m pip install pygame
py -3.11 main.py
now enter desired grid size
How to control it:
Add walls through a mouse click on whichever block you want, else add random walls on the grid by pressing 'M' on the keyboard,
'R' is for reset
1 is for A* Algorithm and 2 is for Greedy Best First
By clicking on Space bar your algorithm will run
Press 'H' to switch to euclidean from Manhattan
